UserTag
UserTagging allows you to notify a user when you quote(mention) them in a post.

Only the first 5 users quoted can be notified per post.

Notified users are sent a PM with a link and short summary of the post you tagged them in.

If you wish to never receive alerts from UserTagging, request to join the UserTag Blacklist HERE.
